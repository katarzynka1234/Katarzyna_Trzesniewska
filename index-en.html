About me

My name is Katarzyna Trześniewska. Economist by education, while Front-end Developer by passion. Fascinated with the visual possibilities of current web technologies. Fun of cross-country skiing and board games lover.

Development in all areas of my life is a source of satisfaction for me. I’m aware that I still have a lot of learning to do, but that's gives me a lot of motivation to work.

Currently I live in Jelenia Gora. I would like to start as a Junior Front-end Developer.


A year ago I decided to change my career path towards IT. I decided to move to Front-end, because I'm sensitive to the aesthetics and graphic form of the content presented on the web. I also like to develop the visual effects of my projects.

In May of this year I completed an intensive Front End Developer bootcamp in Kodilla, run by Codemy S.A. in Wrocław. This three-month online course gave me a solid basis and allowed me to learn the currently used web technologies. I like very much CSS3, which with Sass offers great opportunities for creating styles and animations. Learning JavaScript and jQuery opened me a Word of scripting languages, which inspired me a lot. I am going to explore it. I'm currently learn Angular.

I work also with: Visual Studio Code, version control system Git and NPM: task runner including creation of project structure, tests of code, compilation of SASS on CSS and adding necessary prefixes.

Programming has become one of my passions. Creating web applications and giving them beauty and the possibility to interact with the user get me a lot of fun. I always try to look for the solutions of the problems. This allows me to discover completely new possibilities and it gives me the greatest satisfaction.

Below I presented my projects. I still develop and improve them. I encourage you to see them. If you have any comments, please contact me!

My projects

Slice JPG to HTML

The project was made as part of a bootcamp. Self-programmed template in HTML and CSS based on the received JPG file. The possibilities used here include Flexbox, Transition and pseudo-clasess. The website is RWD thanks to the use of Media queries.

Slice PSD to HTML

Template based on a PSD file. Self—programmed project in HTML and CSS template with use Sass, Bootstrap, JS and jQuery. Used possibilities of Sass: variables, nesting and mixins. Bootstrap’s carousel and grid. JS and jQuery made it possible to get interactivity of selected buttons and animation of icons. RWD made thanks to Media queries and Bootstrap’s grid. The project has a Git version control.

Form reservation

The project allow me to practice the ability to create forms using Bootstrap and jQuery. The form has validation on the filling side. The project is RWD. Styles created by compiled Sass to CSS. The project has a version control system Git. .

Stone, Paper, Scissors

Application for playing the popular game. The game is programmed as part of a bootcamp with using Bootstrap and JavaScript. The project allowed me to practice JS with creating variables, conditional instruction if, swich and defining functions as well as manipulation of DOM elements.

Kanban Board

Kanban Board is used to manage tasks and projects. The project has been programmed during the bootcamp. The application uses jQuery and jQueryUI. Layout and graphics designed by myself with Sass compiled to CSS. The project allow me to learn the basics of communication with the server by Ajax and API.
