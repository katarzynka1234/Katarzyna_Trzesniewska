O mnie

Nazywam się Katarzyna Trześniewska. Z wykształcenia ekonomistka, z zamiłowania Front-end Developer. Zafascynowana możliwościami wizualnymi aktualnych technologii webowych. Amatorka biegów narciarskich oraz miłośniczka gier planszowych.

Ciągły rozwój we wszystkich płaszczyznach życia jest dla mnie źródłem satysfakcji. Jestem świadoma, że przede mną jeszcze wiele nauki ale właśnie to daje mi ogromną motywację do działania.

Aktualnie mieszkam w Jeleniej Górze. Chciałabym rozpocząć pracę jako Junior Front-end Developer.


Rok temu podjęłam decyzję o zmianie ścieżki rozwoju zawodowego w stronę IT. Zdecydowałam się pójść w kierunku Front-end, ponieważ jestem wrażliwa na estetykę oraz formę graficzną prezentowanych w sieci treści. Lubię również rozwijać wizualne efekty swoich projektów.

W maju tego roku ukończyłam intensywny bootcamp ‘Front-end Developer’ w Kodilli. Trzymiesięczny kurs dał mi solidne podstawy oraz pozwolił poznać aktualnie stosowane technologie webowe. Najlepiej odnajduję się w języku CSS3, który w połączeniu z SASS oferuje ogromne możliwości tworzenia stylów oraz animacji. Poznanie JavaScript oraz jQuery otworzyło przed mną świat języków skryptowych, który mnie bardzo zainspirował. Mam zamiar go coraz bardziej zgłębiać. Aktualnie poznaję i wprowadzam do projektów AngularJS.

Moje środowisko developerskie to aktualnie: Visual Studio Code (pracowałam również na SublimeText3); system kontroli wersji Git; NPM: task runner obejmujący automatyczne tworzenie struktury projektu, testy kodu, kompilację SASS na CSS oraz dodanie niezbędnych prefixów.

Programowanie stało się jedną z moich pasji. Tworzenie aplikacji webowych i nadawanie im piękna oraz możliwości interakcji z użytkownikiem sprawia mi dużo radości. Zawsze szukam rozwiązań napotkanych problemów, dzięki czemu odkrywam całkiem nowe możliwości i to daje największą mi satysfakcję.

Ponieżej zaprezentowałam swoje dotychczasowe projekty, które w dalszym ciągu rozwijam i doskonalę. Zachęcam do ich przejżenia. Jęśli masz jakieś uwagi skontaktuj się ze mną!

Wykonane projekty

Cięcie szablonu JPG do HTML

Projekt został wykonany w ramach bootcampu. Samodzielnie zaprogramowany szablon w HTML i CSS na podstawie otrzymanego pliku JPG. Wykorzystane tu możliwości to m.in. Flexbox, Transition, pseudoklasy. Strona jest responsywna dzięki użyciu Media queries.

Cięcie szablonu PSD do HTML

Szablon wykonany na podstawie otrzymanego pliku w formacie PSD. Samodzielnie zaprogramowany w HTML i CSS szablon przy użyciu Sass, Bootstrap, JS oraz jQuery. Wykorzystane możliwości Sass to: variables, nesting oraz mixins. Carousel oraz Grid z Bootstrapa. JS oraz jQuery pozwoliły uzyskać interaktywność wybranych przycisków oraz animacji ikon. Responsywność strony uzyskana dzięki Bootstrap Grid oraz Media queries. Projekt wykonany z kontrolą wersji Git.

Formularz rezerwacji

Projekt służący przećwiczeniu możliwości tworzenia formularzy z użyciem Bootstrap oraz jQuery. Responsywny z bieżącą walidacją wypełnienia pól. Style CSS utworzone za pomocą preprocesora Sass. W projekcie prowadzony jest system kontroli wersji.

Szablon w Bootstrap 4

Projekt jest wykonany za pomocą Bootsrap 4. Style oraz pozycjonowanie elementów szablonu jest uzykane głównie za pomocą klas Bootsrapa. W projekcie utworzono tylko kilka własnych klas w Sass. RWD utworzone za pomocą nowego systemu grid Bootstrapa opartego o Flexbox.

Kamień, Papier, Nożyce

Aplikacja do popularnej gry. Aplikacja napisana w ramach bootcampu z wykorzystaniem Bootstrapa oraz JavaScript. Projekt pozwolił mi przećwiczyć tworzenie zmiennych, instrukcji warunkowych if, swich oraz definiowanie funkcji a także manipulację elementami DOM.
