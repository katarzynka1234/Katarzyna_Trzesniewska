O mnie

Nazywam się Katarzyna Trześniewska. Z wykształcenia ekonomistka, z zamiłowania Front- End Developer. Zafascynowana możliwościami wizualnymi aktualnych technologii webowych. Amatorka biegów narciarskich oraz miłośniczka gier planszowych.

Ciągły rozwój we wszystkich płaszczyznach życia jest dla mnie źródłem satysfakcji. Jestem świadoma, że przede mną jeszcze wiele nauki ale właśnie to daje mi ogromną motywację w działaniu.

Aktualnie mieszkam w Jeleniej Górze. Chciałabym rozpocząć pracę w tym kierunku jako Junior Front-end Developer.


Rok temu podjęłam decyzję o zmianie ścieżki rozwoju zawodowego w stronę IT. Zdecydowałam się pójść w kierunku Front-end, ponieważ jestem wrażliwa na estetykę oraz formę graficzną prezentowanych w sieci treści. Lubię również rozwijać wizualne efekty swoich projektów.

W maju tego roku ukończyłam intensywny bootcamp ‘Front-end Developer’ w Kodilli. Trzymiesięczny kurs dał mi solidne podstawy oraz pozwolił poznać aktualnie stosowane technologie webowe. Najlepiej odnajduję się w języku CSS3, który w połączeniu z SASS oferuje ogromne możliwości tworzenia stylów oraz animacji. Poznanie JavaScript oraz jQuery otworzyło przed mną świat języków scryptowych, który mnie bardzo zainspirował. Mam zamiar go coraz lepiej zgłębiać. Aktualnie poznaję i wprowadzam do projektów AngularJS.

Moje środowisko developerskie to aktualnie: Visual Studio Code (pracowałam również na SublimeText3); system kontroli wersji Git; NMP: tusk runner obejmujący automatyczne tworzenie struktury projektu, testy kodu, kompilację SASS na CSS oraz dodanie niezbędnych prefixów.

Programowanie stało się jedną z moich pasji. Tworzenie aplikacji webowych i nadawanie im piękna oraz możliwości interakcji z użytkownikiem sprawia mi dużo radości. Zawsze szukam rozwiązań napotkanych problemów, dzięki czemu odkrywam całkiem nowe rozwiązania i to daje największą satysfakcję.

Ponieżej zaprezentowałam swoje dotychczasowe projekty, które w dalszym ciągu rozwijam i doskonale. Zachęcam do ich przejżenia. Jęśli masz jakieś uwagi nie wahaj sie ze mną skontaktować!

Wykonane projekty

Cięcie szablonu JPG do HTML

Projekt został wykonany w ramach bootcampu. Samodzielnie zaprogramowany szablon w HTML i CSS na podstawie otrzymanego pliku JPG. Wykorzysane tu możliwości to m.in. Flexbox, Transition, pseudoklasy. Strona jest responsywna dzięki użyciu Media queries.

Cięcie szablonu PSD do HTML

Szablon wykonany na podstawie otrzymanego pliku w formacie PSD. Samodzielnie zaprogramowany w HTML i CSS szablon przy użyciu Sass, Bootstrap, JS oraz jQuery. Wykorzystane możliwości Sass to: variables, nesting oraz mixins. Carousel oraz Grid z Bootsrapa. JS oraz jQuery pozwoliły uzyskać interaktywność wybranych przycisków oraz animacji ikon. Responsywność strony uzyskana dzięki Boostrap Grid oraz Media querre. Projekt wykonany z kontrolą wersji Git.

Formularz rezerwacji

Projekt służący przećwiczeniu możliwości formularzy z użuciem populrnej biblioteki Bootstrap. Responsywny z walidacją wypełnienia pól. Style CSS utworzone za pomocą preprocesora Sass. W projekcie prowadzony jest system kontroli wersji.

Gra w Marynarza

Aplikacja do gry w popularnego Marynarza. Aplikacja napisana w ramach bootcampu z wykorzysaniem Bootrapa oraz JavaScript Projekt pozwolił mi przećwiczyć tworzenie zmiennych, istrukcji warunkowych if, swich oraz definiowanie funkcji a także manimulację elementami DOM.

Kanban Board

Aplikacja Kanban Board służy do zarządzania zadaniami i projektami. Projekt ten został zaprogramowany w ramach bootcampiu. W projekcie jest użyty jQuery oraz jQueryUI. Wykonaie projektu pozwoliło mi poznać podstawy działania komunikacji z serwerem za pomocą Ajax oraz API.
